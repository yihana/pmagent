삼성자산운용 IT 운영 관리 시스템 신규 개발 제안요청서(RFP)

2024. 4. 24

# **1. 프로젝트 개요**

- 프로젝트명: 삼성자산운용 IT 운영 관리 시스템 신규 개발
- 대상용역: 기획 / 디자인 / 개발(PC)
- 사업기간: ’24년 5월 27일(월) ~ ’24년 11월 29일(금) (6개월)
- 입찰방식: 경쟁입찰

# **2. 프로젝트 지향점**

“사내 IT 운영 및 관리를 수용할 수 있는 통합 시스템 구축”

주요 프로세스 흐름:

① 예산 수립

→ ② SR 신청

→ ③ 품의(전자결재)

→ (구매 or 프로젝트 진행)

→ ④ 자산 등록(자산화)

지향점 상세:

- 최신 UI/UX Skill을 활용한 심플하고 동적(Interactive) 디자인
- 업무 담당자 니즈 반영 + 전문가적 아이디어 제공

# **3. 프로젝트 수행 범위**

(※ 예상 Page 본수 및 기능은 별첨 1 ‘ITBM 작업 요건’ 참고)

**① 메인 페이지(Main)**

- IT 업무 포털 역할
- 메인 배너, 공지사항, IT 콘텐츠 업로드
- 각 IT 프로세스로 가는 Quick Link 제공

**② IT SR (ISR)**

- IT SR 요청 기능
- 작업요청에 맞는 Knox 그룹 포탈 결재 상신 기능
- 결재 후 담당자 지정
- 작업 현황 업데이트 및 메일 공유

**③ IT Asset (AMS)**

- 구매·개발 완료 자산 등록 및 관리 (소유자, 위치 포함)
- IT SR 신청 기반 구입일 경우, 품의(CMS) + SR(ISR) 내용 연동 조회

**④ IT Budget (BMS)**

- 매년 9월 정기 예산(개발·비용) 신청/등록/조회
- 예산 변동 시 기록 관리
- 비용 예산 등록 시 Excel 일괄 업로드
- 품의(CMS) 승인 정보 기반 금액 자동 반영
(승인금액, 승인잔액, 지급금액, 지급잔액 등)
- 사용 내역 기록

**⑤ IT Contract & Approval (CMS)**

- 계약 / 실시 / 정산 / 지급 / 수정 품의 + 보고서 작성·결재·조회
- Knox 포탈 API 연동 결재
- 실시품의 시 비용 및 지급 횟수 기반 지급 데이터 생성
- 지급 데이터 기반 정산품의 가능하도록 구성

**⑥ IT Project (IPS)**

- 프로젝트 유관자 등록
- 프로젝트 진행사항 및 요청사항 기록
- ISR의 요청(외주 인력 등록, PC 세팅 등)을 별도 태그로 관리
- 프로젝트 관련 정보 기록
(방화벽, 서비스 인스턴스, 라이선스, 계정 등)
- 프로젝트 완료 후 보안 검수 및 완료 품의
- 종료 후 유지보수 여부 기록 (연 단위 유지보수 계약 판단)

**⑦ 공통 사항**

- 데이터 변경 내역 조회 및 Excel 다운로드

| **구분** | **내용** |

# **6. 업체 선정 방법**

1. 내부 규정 및 평가표에 따라 선정
2. 평가 기준·방법은 전적으로 당사 고유 권한

**평가 중점 고려사항**

**비가격 평가 (70점)**

- IT 운영·관리 시스템 기획 능력(20)
- 요구사항 이해·반영 능력(15)
- 최신 기술·지식 구현 능력(15)
- 프로젝트 참여 인력 업무 자세(10)
- 유사 프로젝트 수행 경험(5)
- 전문가적 아이디어 제공(5)

**가격 평가 (30점)**

- 제안금액 상대평가(15)
- SI 엔지니어 투입 단가 협상력(10)
- 정확한 스콥 및 예산 산출 역량(5)
